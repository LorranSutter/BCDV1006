Client / server

A client any type of user that access a service in the web. This service is provided by another computer called server.

Client server model is system that establish communication between servers and clients through computer network (source: medium).

HTML tags

Tags are hidden keywords used to organize and define where the elements will be in a web page (source: simplehtmlguide).

There are lots of different tags for different purpose, such as img to index an image or body to define the main part of the web page.

Git

Git is a version control system widely used among programmers to manage their codes (source: wikipedia).

It ensures that every modification will be saved in the history of git so as to blame some workmate for wrong code.
